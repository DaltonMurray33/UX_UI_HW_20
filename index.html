UI Website Design

Department of Homeland Security

Project
Overview

This was my first UI design project from my Ohio State University bootcamp course. I was given a government website to redesign the homepage to make the UI more user friendly.

Problem

The website is hard to maneuver and simple tasks are hard to complete.

Solution

Redo the header, homepage, and footer to make the website more user friendly.

Tools

  • Figma / Figjam
  • Google Doc/Sheets
  • Miro

My Role

  • Ux Research
  • UI Design
  • User Testing
  • Wireframing

Timeline

  • Overall: 3 Weeks
  • Discory & Research: 1 Week
  • Design & Testing: 2 Weeks

My Design Process

Interviews

I built a Proto Persona based on the user interviews I conducted. I presented them with the same pain points and challenges that my users had.

Typical User Path

Preparing for user test on the current website to see what needs to be fixed to make the design more user friendly. I planned guerrilla testing with 3 simple task for the user to complete.

2x2 Matrix

Based off the results of the guerrilla testing I set up a 2x2 matrix to show the important of the changes that need done.

Hueristic Evaluation

Next, I conducted a heuristic evaluation on the homepage to give myself a better idea of what frustrations the users faced with the drop-down menu, tabs, and footer.

Moodboard

I made a moodboard to show the UI colors and design identity

Card Sorting

I used card sorting to clean up the header menu along with the footer. I found during the user testing that a lot of topics didn't belong in their current location.

Sitemap

Once I finished card sorting I made a sitemap to ensure seamless navigation and to make the information easier to navigate.

UI Style Tile

I made a style tile to communicate the core of the visual brand.

Style Guide

I used a style guide to show all the design elements and interactions that occur within the site.

Learnings

This was my first UI project so I learned a ton! Basically everything in this case study was new to me in the moment. The most valuable thing I learned is to always always always use the sizing templates! Also when making the prototypes clickable components are amazing time savors.
